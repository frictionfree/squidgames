Welcome To Squid Coding Games

Implement solutions to simple Python code quizes to your liking and immediately get your relative benchmark.

Getting Started
  • Select one out of the available coding quizes
  • Clone the SquidGameClient repo on GitHub
  • Install the required dependencies using pip install
  • In games directory, identify the relevant py file implementing the client for the game you selected
  • Implement just the def play_game(game_params : json) class function
  • Run the Squid Game Client with command line paramters:
      • game_client.py --username *** --password *** -game SumQwest
